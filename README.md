# UCI Parser

Scrape Chess.com and generate markdown tables using this UCI (Universal Chess Info) parser

## Installation

### If you have Git installed:

- Open CMD
- Go to preferred location
- Run git clone https://github.com/SiegieSiegie/uci-parser

### If don't:

- Download code ZIP
- Unpack wherever

## Usage

Run `cli.bat` and follow instructions from the terminal

## Tournament Names
